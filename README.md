# Dark.Chat Frontend

Это фронтенд часть веб-приложения для обмена сообщениями Dark.Chat.

## ✨ Основные возможности

*   Обмен сообщениями в реальном времени
*   Аутентификация пользователей (регистрация и вход)
*   Личные и групповые чаты
*   Отправка файлов и изображений
*   Статусы пользователей (онлайн/офлайн)
*   Уведомления о новых сообщениях

## 🛠️ Стек технологий

*   **Framework:** React
*   **Language:** TypeScript/JavaScript
*   **Real-time Communication:** WebSockets
*   **Build Tool:** Vite

## 🚀 Установка и запуск

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Dark-Tehno/Dark.Chat.git
    cd Dark.Chat 
    ```

2.  Установите зависимости:
    ```bash
    npm install
    ```

3.  Запустите проект в режиме разработки:
    ```bash
    npm run dev
    ```
    Приложение будет доступно по адресу `http://localhost:5173` (или другому порту, указанному в конфигурации).
